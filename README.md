# Reminder Backend

Учебный проект в виде приложения для напоминаний, реализованный на **Java Spring Boot**.

Приложение предоставляет REST API для управления напоминаниями пользователей: создание, редактирование, удаление, поиск, сортировка, фильтрация и вывод списка с пагинацией.

> В репозитории реализована только **backend-часть**.

---

## Функциональность

- создание напоминания;
- редактирование напоминания;
- удаление напоминания;
- получение списка напоминаний;
- поиск по названию, описанию, дате и времени;
- сортировка по названию, дате и времени;
- фильтрация по дате и времени;
- пагинация;
- отправка уведомлений по расписанию.
